EX5: CI Pipeline: Python App → GitHub → Docker Hub
app.py
print("Hello CI Pipeline from Reshma!")
Dockerfile
FROM python:3.11
WORKDIR /app
COPY . .
CMD ["python", "app.py"]
Step 4 — Build and test locally
docker build -t reshmatina12/demo:local .
docker run reshmatina12/demo:local
 You should see:
Hello CI Pipeline from Beereddy!
Step 5 — Login and push manually (optional)
docker login
docker push reshmatina12/demo:local
mkdir .github/workflows/main.yml
name: Docker Build & Push
on: [push]
jobs:
 build:
 runs-on: ubuntu-latest
 steps:
 - uses: actions/checkout@v4
 - uses: docker/login-action@v3
 with:
 username: ${{ secrets.DOCKERHUB_USERNAME }}
 password: ${{ secrets.DOCKERHUB_TOKEN }}
 - uses: docker/build-push-action@v6
with:
 push: true
 tags: reshmatina12/demo:latest
Initialize git and commit
git init
git add .
git commit -m "initial commit"
git branch -M main
git remote add origin https://github.com/reshmatina12/demo.git
git push -u origin main
Step 9 — Add GitHub Secrets
Go to:
GitHub → demo repo → Settings → Secrets → Actions
Add two secrets:
Secret Name Value
DOCKERHUB_USERNAME reshmatina12
DOCKERHUB_TOKEN
(create token in Docker Hub → Account Settings → Security →
New Access Token)
 Step 10 — Done
Push any code change:
git add .
git commit -m "update"
git push
Then go to your GitHub repo → Actions tab → see the workflow run.
After success, check Docker Hub →
docker build -t reshmatina12/demo:local .
docker push reshmatina12/demo:local
